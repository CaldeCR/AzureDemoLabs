A- Azure DDos Standard protection
B- Azure Firewall
C- Application Gateway with WAF

Correct 
Option D: Network Security Group (NSG)
The Logic: An NSG is the most basic and fundamental security tool for an Azure Virtual Network (VNet). It acts as a simplified firewall that uses rules to allow or deny inbound and outbound traffic based on source/destination IP address, port, and protocol.

A- By copying several GBs of data to Azure from an on-premises network over a VPN, additional data transfer costs are incurred

Correct:

Option C.
When using a general-purpose v2 Azure Storage account, you are charged based on two main factors: the amount of data stored and the number of read and write operations performed. Data transfer costs in Azure are affected by network egress. Copying several gigabytes of data from Azure to an on-premises network over a VPN results in outbound data transfer charges. This is a valid cost factor in Azure, so this statement is correct.
